# Minimal Skill

**Description:** A minimal demonstration skill showing the simplest possible skill structure.

**When to use:** When you need to demonstrate basic skill functionality.

## Workflow

1. Acknowledge that you've loaded the minimal skill
2. Explain that this is the simplest possible skill structure
3. Show that you can access the skill's content

## Key Points

- Skills must have a description
- Skills can define workflows
- Skills load progressively (only when needed)
- This skill has no references, scripts, or assets

## Example Usage

When someone asks "show me the minimal skill", respond with:
- Confirmation that the minimal skill is loaded
- Explanation of what makes it minimal
- Its structure and purpose
